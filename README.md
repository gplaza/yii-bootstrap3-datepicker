Yii-bootstrap3-datepicker
=========================

Datepicker using bootstrap 3

Usage
-----

```
<?php $this->widget('ext.datepicker.DatePickerControl', [
    'model' => $model,
    'attribute' => 'birth_date',
    'options' => [
        'placement' => 'right',
        'autoclose' => true,
        'todayBtn' => true,
    ]
]); ?>
```

Have fun!
